Get Started

To use Vibey, you'll need an OpenRouter API key. OpenRouter gives you access to many AI models including Claude, GPT-4, and Gemini.

  1. Go to openrouter.ai/keys
  2. Create a free account
  3. Generate an API key
  4. Paste it below

Get your key at openrouter.ai/keys

Your API key is stored only in your browser's local storage and is never sent to any server except OpenRouter.

Browse available models at openrouter.ai/models

What this does: Replaces old code versions with placeholders, sending only the latest code to the model.

Trade-off: Reduces token count but breaks prompt caching. Models with caching (most providers) benefit more from sending full history.

Use this for: Models without caching support, or very long conversations where token limits are a concern.
