Get Started

To use Vibey, you'll need an OpenRouter API key. OpenRouter gives you access to many AI models including Claude, GPT-4, and Gemini.

  1. Go to openrouter.ai/keys
  2. Create a free account
  3. Generate an API key
  4. Paste it below

Get your key at openrouter.ai/keys

Your API key is stored only in your browser's local storage and is never sent to any server except OpenRouter.
